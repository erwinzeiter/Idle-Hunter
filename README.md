# Idle Hunter

A Godot 4.x idle/incremental game inspired by Idle Slayer.

## Quick Start

1. Open the project in Godot 4.3 or later
2. Press F5 to run
3. Click "Test Kill Enemy" to earn souls
4. Purchase upgrades to progress

## Features

- **Clean Architecture:** GameState as single source of truth
- **Signal-Based Communication:** Decoupled systems
- **No UI Logic:** All game logic in dedicated systems
- **Auto-Save:** Game saves every 60 seconds
- **Offline Progress:** Earn passive income while away (up to 8 hours)
- **Scalable Design:** Easy to add new features
